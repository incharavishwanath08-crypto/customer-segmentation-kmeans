# Customer Segmentation & Behavioral Analysis Using K-Means Clustering

## Project Overview

This project applies K-Means Clustering, an unsupervised machine learning technique, to segment customers into distinct groups based on their demographic and spending behavior. The goal is to identify meaningful customer segments that can help businesses improve marketing strategies, customer engagement, and decision-making.

## Problem Statement

Businesses often have diverse customer bases with varying purchasing behaviors. Treating all customers the same can lead to ineffective marketing efforts. This project aims to identify customer groups with similar characteristics and spending patterns using clustering techniques.

## Dataset

The dataset contains customer-related information such as:

- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

The data is used to discover hidden patterns and create customer segments.

## Project Workflow

### 1. Data Preprocessing
- Data inspection and cleaning

- Feature selection

### 2. Exploratory Data Analysis (EDA)
- Customer demographic analysis
- Income distribution analysis
- Spending behavior analysis

### 3. K-Means Clustering
- Applied K-Means clustering algorithm
- Determined optimal number of clusters using the Elbow Method
- Grouped customers into meaningful segments

### 4. Data Visualization
- Cluster visualization using scatter plots
- Customer segment comparison
- Distribution analysis of each cluster
